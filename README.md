# GesstIA

> Gestión SST inteligente — Centraliza tu cumplimiento normativo en un solo lugar.

## ¿Qué es?

GesstIA es una plataforma web pensada para ayudar a empresas chilenas a cumplir con la **Ley 21.719 de Protección de Datos Personales** (vigente 1 diciembre 2026) y la normativa **SST chilena** (DS 44, Ley 16.744, PREXOR, TMERT, etc.).

### Herramientas incluidas

| Archivo | Descripción |
|---|---|
| `index.html` | Landing page + plan de precios + diagnóstico de 15 preguntas |
| `gesstia_checklist.html` | Checklist de 30 puntos para medir avance de adecuación |
| `gesstia_rat_generator.html` | Generador de Registro de Actividades de Tratamiento (Ley 21.719) |

Todas son **archivos HTML autónomos** — no requieren servidor, base de datos ni instalación. Funcionan directo en el navegador.

## Cómo usar

1. Descargar cualquier archivo `.html`
2. Abrir en el navegador
3. Los datos ingresados se guardan localmente (no se envían a ningún servidor)
